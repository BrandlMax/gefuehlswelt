# Gefuehlswelt

## Install (once):
1. Install Yarn
``` brew install yarn ```
2. Install Cordova
``` npm install -g cordova ```
3. Install Phonegap
``` npm install -g phonegap ```
4. Update Dependencies
``` yarn install ```


## Start Working:
Then Open Gefuehlswelt Folder in new Terminal-Window and Enter:
``` yarn devmode```

Open Gefuehlswelt Folder in Terminal and Enter:
``` phonegap serve ```
